pycharm pyhton模板代码

最终样例
```python
# -*- coding:utf-8 -*-
# @time :2020/7/29
# @IDE : pycharm
# @author :lxztju
# @github : https://github.com/lxztju
```


配置方式

`file-->setting-->Editor--->File and Code Templates`

然后输入：

```python
# -*- coding:utf-8 -*-
# @time :${DATE}
# @IDE : pycharm
# @author :lxztju
# @github : https://github.com/lxztju
```

可以自己更新补充信息
